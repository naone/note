### Heading
```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

### Teks miring
```
*miring*
```

### Teks tebal
```
**Tebal**
```

### Coret
```
~~ Coret ~~
```

### List
#### ordered list
```
1. list 1
2.list 2
 1. ordered sub-list
```
#### unordered list
```
- list 1
- list 2
 - unordered sub list
```

### Gambar
```
![alt text](link ke gambar)
```

### Code dan syntax highlight
#### inline code
```
`code`
```
#### Block code
```
```Block code
    linuxisekai```
```

### Link
```
[text link](url link)
```

### Tabel
```
| Tabel Heading 1 | Tabel Heading 2 | Tabel Heading 3 |
| --------------- | --------------- | --------------- |
| baris 1 kolom 2 | baris 2 kolom 2 | baris 3 kolom 2 |
| baris 1 kolom 3 | baris 2 kolom 3 | baris 3 kolom 3 |
```

### Blockquotes
```
> Blockquotes
> Masih bagian dari block qoutes diatas
```
