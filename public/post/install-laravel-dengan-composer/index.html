Install Laravel Dengan Laravel Installer
July 25, 2019 || tempegoreng

install composer dan laravel installer

sudo pacman -S composer
composer global require laravel/installer

perintah diatas akan membuat folder composer directory .config.

export laravel ke zsh/bash agar bisa dieksekusi

export PATH=$PATH:$HOME/.config/composer/vendor/bin

membuat project laravel

laravel new project_laravel

local development server

php artisan serve
